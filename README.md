# CleanTube: Hide YouTube Shorts 🛡️🚫

CleanTube is a premium Chrome Extension (Manifest V3) designed to help you regain focus by completely removing YouTube Shorts from your browsing experience.

## ✨ Features
- **Elite Dashboard**: A sleek, modern UI with real-time controls.
- **Granular Hiding**:
  - Hide the Sidebar "Shorts" button.
  - Clear "Shorts" shelves/vines from your Home feed.
  - Instantly redirect direct Shorts URLs to the standard YouTube Home.
- **High Performance**: Uses Chrome's `declarativeNetRequest` for engine-level redirection.
- **Privacy Focused**: No data collection, minimal permissions.

## 🛠️ Installation (Manual/Developer Mode)
1. Download this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer Mode** (top right).
4. Click **Load unpacked** and select the extension folder.
5. Pin the extension to your toolbar.
